# 首页重设计：语画姬 极简落地页

**日期：** 2026-04-03  
**范围：** `frontend/src/views/Landing.vue` + 相关 i18n 文字  
**目标：** 去除花哨视频动效，换成清新明亮的极简风格，聚焦公告板 + 使用指南 + API 文档

---

## 1. 背景与目标

当前首页（`Landing.vue`）大量依赖即梦 CDN 视频素材，品牌硬编码为"小野 AI"，动效繁重。需要改造为自用网站"语画姬"的落地页，风格清新简洁，内容聚焦开发者/用户快速上手。

---

## 2. 整体风格

| 属性 | 值 |
|------|----|
| 背景 | 白色 `#FFFFFF`，区块间隔用浅灰 `#F8F9FA` |
| 主题色 | 柔和紫色 `#7C3AED`（按钮、强调色、分割线） |
| 字体 | 系统字体栈（无需引入外部字体） |
| 动效 | 无（去掉所有 IntersectionObserver 滚动动画、视频轮播） |
| 主题模式 | 强制浅色（移除 `forceTheme('dark')`） |
| 外部依赖 | 无（去掉所有即梦 CDN 视频/图片引用） |

## 4. 各区块详细设计

### 4.1 导航栏

- 左：`/images/icon.png` 图标 + "语画姬"文字
- 右：一个"进入平台"主按钮 → `router.push('/inspiration')`
- 无多余导航链接，无语言切换（落地页固定中文）
- 白底，底部 1px 浅灰边框

### 4.2 公告板

- 白色卡片，最大宽度 760px，居中
- 顶部"📢 公告"标题行
- 内容：静态数组，每条包含 `{ date, title, content }`
- 初始展示最新 1 条，若有多条可展开
- 暂无动态接口，内容硬编码在组件里，后续可改为接口驱动

### 4.3 使用步骤

- 横向 3 列卡片（移动端纵向堆叠）
- 每张卡片：序号圆圈 + 步骤标题 + 简短说明
  - **Step 1** — 注册账号：访问平台，邮箱注册
  - **Step 2** — 获取密钥：登录后兑换密钥，或充值积分
  - **Step 3** — 开始创作：在生成页输入提示词，选择模型

### 4.4 API 文档

展示 3 个核心接口，每个接口用折叠卡片或分区块展示：

#### `POST /api/auth/login`
- 说明：邮箱登录，返回 JWT Token（后续接口均需携带）
- 参数：`email`（string）、`password`（string）
- 返回：`{ token: "..." }`

#### `GET /api/models`
- 说明：获取当前可用的模型列表及定价
- 无需认证

#### `POST /api/generate`
- 说明：统一生成接口，支持图像/视频
- Header：`Authorization: Bearer <token>`
- 参数：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | string | 是 | `image` 或 `video` |
| `prompt` | string | 是 | 文字描述 |
| `model` | string | 否 | 模型 ID，默认使用推荐模型 |
| `params` | object | 否 | 额外参数，如 `{ "size": "1K" }` |

- curl 示例：
```bash
curl -X POST https://your-domain/api/generate \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"type":"image","prompt":"一只橘猫坐在窗台上","model":"nanobanana-pro"}'
```
